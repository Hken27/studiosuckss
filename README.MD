# 🌐 Website Portofolio – Studio & Layanan Pengembang

Website ini merupakan portofolio profesional yang dirancang untuk menampilkan berbagai **proyek studio kreatif** dan **layanan pengembangan serta maintenance program**. Website ini menyatukan tampilan visual yang modern dengan informasi yang lengkap, cocok untuk pelaku industri kreatif dan teknologi.

## 🎯 Tujuan

Memberikan gambaran profesional tentang:
- Produksi foto dan video dari studio kreatif
- Proyek-proyek pengembangan aplikasi (web & mobile)
- Layanan pemeliharaan dan support sistem digital

## 🚀 Fitur Utama

- ✅ **Halaman Hero Dinamis** – Menampilkan animasi dengan teks berjalan
- ✅ **Responsif** – Tampilan optimal di semua perangkat: HP Android, iPhone, tablet, laptop (Windows & MacBook)
- ✅ **Tentang Kami** – Penjelasan tentang tim dan layanan
- ✅ **Form Kontak** – Formulir untuk menghubungi tim secara langsung
- ✅ **Struktur Modular** – Mudah dikembangkan dan dimodifikasi

## 🛠️ Teknologi yang Digunakan

- HTML | CSS | JavaScript dasar (untuk efek interaktif)
- Assets media (gambar, GIF)
